## üìÇ Other Academic Projects

Additional work demonstrating versatility in Computer Science (CS) foundations and Mathematical Modeling.

* **[Social Network Graph](https://github.com/IvanMV01/Data-Science-Portfolio/tree/main/Social%20Network%20Graph) (Haskell):**
    * Modeled a social graph (Users, Relationships, Posts) using **pure functional programming**.
    * Implemented **recursive algorithms** for network queries, such as pathfinding between users (`existeSecuenciaDeAmigos`) and complex logic predicates ("loyal follower" logic).
* **[Input-Output Matrix Analysis](https://github.com/IvanMV01/Data-Science-Portfolio/tree/main/Input-Output%20Matrix%20Analysis) (Python/Math):**
    * Computational Linear Algebra applied to Economic Modeling (**Leontief**).
    * Implemented core algorithms from scratch: **LU Decomposition**, **Power Method** (Eigenvalues), and **PCA** to simulate economic shocks on real-world data (CEPAL).
